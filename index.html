Data Pipeline for Earnings Dashboard


/**
 * Simplified ETL pipeline for the earnings dashboard
 * (illustrative only, not production code)
 */
async function buildEarningsDataset(qtr) {
  const revenue = await fetchKPI("revenue", qtr);
  const opex    = await fetchKPI("operating_expenses", qtr);

  const netIncome = revenue - opex;
  const margin = netIncome / revenue;

  return {
    quarter: qtr,
    revenue,
    opex,
    netIncome,
    margin
  };
}

Key Financial Formulas

Return on Equity (ROE): \[ \text{ROE} = \frac{\text{Net Income}}{\text{Shareholders' Equity}} \]

Net Interest Margin (NIM): \[ \text{NIM} = \frac{\text{Interest Income} - \text{Interest Expense}} {\text{Average Earning Assets}} \]

Cost-to-Income Ratio (CTI): \[ \text{CTI} = \frac{\text{Operating Expenses}}{\text{Operating Income}} \]

Use the formulas to tie back to earlier KPIs. - Show how improving income or controlling costs affects CTI and ROE. - Emphasize that these ratios are closely watched by regulators and investors.
